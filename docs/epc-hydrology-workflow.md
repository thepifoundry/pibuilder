# EPC Hydrology Workflow

Infrastructure projects require hydrological analysis to determine design flood levels.

Typical EPC workflow:

1. Identify project site.
2. Determine upstream catchment.
3. Collect rainfall data.
4. Perform rainfall frequency analysis.
5. Compute rainfall intensity for design return period.
6. Estimate runoff and flood discharge.
7. Estimate High Flood Level (HFL).
8. Produce hydrology report for design approval.
